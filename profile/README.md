## About

AlpWare Studio is an independent software studio focused on building modern, privacy-first and user-centric mobile applications.

Our mission is simple:

> Create reliable software that respects users, protects privacy and solves real-world problems.

---

## Products

### Adaptive-Hz

Automatic refresh rate control for unsupported Android devices.

- Samsung Galaxy support
- Xiaomi support
- Open Source

### PasswordBook

Offline-first password manager focused on security and privacy.

- Local storage
- No cloud dependency
- Material Design

### Wakeon

Modern Wake-on-LAN solution.

- Multi-device management
- Network discovery
- Flutter

### NotifyBridge

Real-time Android notification bridge for macOS.

- Local network communication
- Privacy-first architecture
