# Contributing

Thank you for your interest in contributing to AlpWare Studio projects.

## Before You Start

Please:

1. Search existing issues first.
2. Open a discussion for major changes.
3. Keep pull requests focused and small.

## Reporting Bugs

When reporting a bug, please include:

- Device information
- Operating system version
- Steps to reproduce
- Expected behavior
- Actual behavior

## Feature Requests

Feature requests should clearly explain:

- The problem being solved
- The proposed solution
- Possible alternatives

## Pull Requests

Before submitting a pull request:

- Follow the project's coding style.
- Keep commits meaningful.
- Test your changes.
- Update documentation if necessary.

Thank you for helping improve our projects.
